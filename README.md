# Quick start

### Install dependencies
#### `npm install`
### Start Node Server
#### `npm start`

# Description
Stage project: créer des listes de course en temps réel.

# Technos
* VueJs
* NodeJs
* Vue Native
* MongoDB

# Dépendances
* Express `npm i express`
* Body-Parser `npm i body-parser`
* Cors `npm i cors`
* Axios `npm i axios`
* Nodemon `npm i nnodemon`
* MongoDB `npm i mongodb`
